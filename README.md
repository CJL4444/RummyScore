# ♣ Rummy Scorekeeper ♠

A clean, browser-based scorekeeper for 2–4 player Rummy games. No installs, no dependencies — just open the HTML file and play.

## Features

- **2–4 players** with custom names
- **Two game modes:**
  - **Set Rounds** — play a fixed number of rounds; lowest total score wins
  - **Max Score** — play until someone hits the losing threshold (you set it)
- Round-by-round score entry with running totals
- Alerts when a player gets dangerously close to the losing score
- End-of-round summary showing high and low scorer each round
- Full scoreboard with per-round history
- Winner screen with final standings

## How to Use

1. Download `rummy-scorer.html`
2. Open it in any modern browser — no server needed
3. Configure your game (players, mode, rounds or score limit)
4. Enter scores after each round and hit **Submit Round**
5. The app tracks everything and announces the winner automatically

## Rules Reminder

In Rummy, **lowest score wins**. Points are counted from cards left in hand at the end of each round. The player who goes out scores zero for that round.

## Tech

Single-file HTML/CSS/JS. No frameworks, no build tools, no dependencies. Works offline.
